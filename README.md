# SparkPost Plugin

Adds support for SparkPost as a mail driver in October CMS. After installing this plugin, SparkPost will appear as a mail driver in the **Settings â†’ Mail Configuration** area.

## Requirements

- October CMS 3.3.8 or above

### Installation

To install with Composer, run this from your project root.

```bash
composer require rainlab/sparkpost-plugin
```
